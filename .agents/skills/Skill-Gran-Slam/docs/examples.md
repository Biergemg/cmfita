# Examples

## Qué cubren los ejemplos
Los ejemplos muestran cómo traducir distintos contextos de negocio a una salida estructurada, sin usar lorem ipsum ni lenguaje inflado.

## Índice
- [`examples/local-services-example.md`](../examples/local-services-example.md): impermeabilización residencial en ciudad con demanda visible y alta confianza requerida.
- [`examples/ecommerce-example.md`](../examples/ecommerce-example.md): marca de skincare con bundle y fricción de recompra.
- [`examples/b2b-services-example.md`](../examples/b2b-services-example.md): consultoría de automatización para despachos contables.
- [`examples/info-product-example.md`](../examples/info-product-example.md): programa formativo para cerrar ventas consultivas.
- [`examples/clinic-or-health-example.md`](../examples/clinic-or-health-example.md): clínica dermatológica con valoración previa y límites de claims.

## Qué mirar en cada ejemplo
- cómo se registran hechos, inferencias y propuestas;
- cómo se marcan supuestos cuando faltan datos;
- cómo cambia el canal principal según nicho y confianza;
- cómo se evita convertir la oferta en una lista genérica de servicios.
