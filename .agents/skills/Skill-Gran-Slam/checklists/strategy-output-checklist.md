# Strategy Output Checklist

Marcar `NO PUBLICAR TODAVÍA` si cualquiera de los siguientes puntos no se cumple.

- [ ] Mercado definido.
- [ ] Cliente objetivo definido.
- [ ] Dolor principal definido.
- [ ] Resultado deseado definido.
- [ ] Temperatura de demanda clasificada.
- [ ] Comparabilidad actual explicada.
- [ ] Ecuación de valor trabajada.
- [ ] Obstáculos y soluciones priorizados.
- [ ] Oferta principal definida.
- [ ] Bonos justificados por fricción, no por adorno.
- [ ] Mecanismo de entrada definido.
- [ ] Garantía definida y prudente.
- [ ] Exclusiones visibles.
- [ ] Lógica de pricing defendible.
- [ ] Canal principal definido.
- [ ] Canal secundario definido o descartado con razón.
- [ ] Lead magnet definido y no genérico.
- [ ] Captura y seguimiento definidos.
- [ ] Activos recomendados priorizados.
- [ ] Plan de implementación por fases.
- [ ] Supuestos explícitos documentados.
- [ ] Huecos de evidencia documentados.
- [ ] Claims no soportados eliminados.
