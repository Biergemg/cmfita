# Anti-Bullshit Checklist

- [ ] No se inventaron testimonios.
- [ ] No se inventaron cifras, benchmarks o tasas de conversión.
- [ ] No se prometieron resultados absolutos fuera de control.
- [ ] No se usó branding bonito como sustituto de oferta.
- [ ] No se bajó precio como respuesta automática.
- [ ] No se escribieron anuncios antes de definir oferta y canal.
- [ ] No se confundió contenido con lead magnet.
- [ ] No se confundió lead magnet con oferta principal.
- [ ] No se recomendó webinar por inercia.
- [ ] No se recomendó `PAID_ADS` con follow-up insuficiente.
- [ ] No se propuso una garantía abierta al abuso.
- [ ] No se escondieron exclusiones, cobertura, tiempos o restricciones.
- [ ] No se presentó inferencia como hecho.
- [ ] No se presentó propuesta como validación cerrada.
- [ ] No se dio por hecho que todos los nichos compran igual.
