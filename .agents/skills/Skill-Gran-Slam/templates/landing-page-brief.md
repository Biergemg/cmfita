# Template — Landing Page Brief

## Propósito
Definir la estructura de una landing solo después de tener oferta, mecanismo de entrada y canal claros.

## Instrucciones de llenado
- No escribir copy ornamental.
- Cada sección debe corresponder a una objeción, fricción o prueba.

## Criterio de completitud
La plantilla está completa solo si la landing tiene objetivo, audiencia, CTA, secciones, pruebas y exclusiones.
